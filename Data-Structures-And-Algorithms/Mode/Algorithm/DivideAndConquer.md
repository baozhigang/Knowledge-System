# 递归算法

分治，典型的递归结构。

### 思想细节

分治算法可以分三步走：分解 -> 解决 -> 合并

1.分解原问题为结构相同的子问题。
2.分解到某个容易求解的边界之后，进行第归求解。
3.将子问题的解合并成原问题的解。
